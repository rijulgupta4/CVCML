# Data Access

The core pipeline was developed with [MIMIC-IV v3.1](https://physionet.org/content/mimiciv/3.1/). Later feasibility and transportability analyses used [eICU-CRD v2.0](https://physionet.org/content/eicu-crd/2.0/) and [ARMD-MGB v1.0.0](https://physionet.org/content/armd-mgb/1.0.0/).

These are credentialed resources. Obtain them directly from PhysioNet after completing the required training and signing the applicable data use agreement.

Keep source tables, clinical notes, extracted cohorts, feature matrices, identifiers, predictions, review queues, adjudication samples, and trained models outside Git and ordinary cloud-synced folders.

PhysioNet states that MIMIC-derived datasets and models should be treated as sensitive and, if shared, distributed through PhysioNet under the same agreement as the source. This repository therefore includes source code, a model card, publication-style aggregate figures, and a narrative report, but no reusable derived result tables.
