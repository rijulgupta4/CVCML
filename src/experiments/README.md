# Experiment Archive

This directory preserves the complete iterative source history. See [`../../docs/RUN_INDEX.md`](../../docs/RUN_INDEX.md) for the authoritative run map.

Runs 1-14 include superseded cohort, reference-time, and feature definitions. They are retained to make leakage corrections and modeling decisions auditable, not as recommended production workflows.

The v0.5 sequence begins with folder `15_v0_5_catheter_episode_redesign` (human-facing Run 16). Generic local path placeholders must be configured before execution. No source data, row-level derivatives, or fitted model artifacts are included.
